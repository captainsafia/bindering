# bindering

![bindering output](https://user-images.githubusercontent.com/836375/33342690-cfc50bf6-d437-11e7-9ae6-b55a66d5288d.png)

To run:

```
git clone https://github.com/rgbkrk/bindering.git
cd bindering
npm install
npm run start
```

This will:

* Launch a binder backed server
* Launch a kernel
* Query kernel status
* Shutdown the kernel
